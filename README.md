## About

GregTech Community Edition is a total re-write of GregTech for newer Minecraft versions. 
It is mostly based off of GregTech 5 Unofficial but with GregTech 6 and GregTech 4 elements thrown in.

### GTCE is not created by nor is it maintained by GregoriusT.

## Downloads

Downloads can be found on the CurseForge site {here} //TODO: add CurseForge project when it's released 

Place the .jar file in your mods folder as you would with any other mod.

## Installation

GTCE requires JEI for its recipes and will not load without it.
